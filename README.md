# Devfolio - Personal Portfolio Project 🚀

A modern, clean, and fully responsive personal portfolio website designed to showcase professional skills and projects.

## ✨ Features
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop screens.
* **Interactive UI:** Includes typing animations and smooth scrolling effects.
* **Modern Design:** Clean layout with a professional look.
* **Optimized Performance:** Fast loading and clean code structure.

## 🛠️ Technologies Used
* **HTML5** & **CSS3**
* **Bootstrap 5** (Grid System & Components)
* **JavaScript** (For animations and interactivity)
